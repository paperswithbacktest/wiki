## What is the consensus mechanism used by Hyperledger Burrow?

Hyperledger Burrow uses a consensus mechanism called Tendermint. This is a way for all the computers on the Burrow network to agree on what transactions should be added to the blockchain. Tendermint is known for being fast and secure, which is important for businesses that need to process a lot of transactions quickly and safely.

In simple terms, Tendermint works by having a group of special computers, called validators, take turns proposing new blocks of transactions. The other validators then check the proposed block and vote on whether it should be added to the blockchain. If most of the validators agree, the block is added. This system helps make sure that everyone on the network trusts the transactions and keeps the blockchain running smoothly.
